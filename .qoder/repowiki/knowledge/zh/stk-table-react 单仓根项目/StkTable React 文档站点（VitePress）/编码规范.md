- 每个功能点拆分为独立的 Markdown 文件，按 `main/table/basic`、`main/table/advanced/custom-cells`、`main/api`、`main/other`、`main/start` 等语义子目录组织，并在对应语言目录下保持完全一致的层级。
- 示例代码采用 Markdown 内联 TypeScript 代码块配合末尾 `<demo react="..." github="...">` 标签双轨展示，前者用于阅读，后者由插件编译为可运行 Demo。
- 多语言站点通过 `locales` 字段集中注册 root/en/ja/ko 四套配置，每套配置复用相同的路由结构，新增语言只需复制一份 `main/` 与 `demos/` 目录。
