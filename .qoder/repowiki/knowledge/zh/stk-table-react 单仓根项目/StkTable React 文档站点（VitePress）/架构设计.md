- 站点根目录 `docs-src/` 为 VitePress 源码根，入口配置位于 `.vitepress/config.ts`，通过 `defineConfig` 声明 `base: '/stk-table-react'`、默认暗色主题与本地搜索。
- 多语言通过 `locales.root/en/ja/ko` 四个分支实现，各语言下复制相同的 `main/`（API、table/basic、table/advanced、other、start）与 `demos/` 结构，`index.md` 作为各语言首页。
- 示例代码以 Markdown 内嵌 `<demo react="..." github="...">` 标签形式编写，由 `vitepress-demo-plugin` 从仓库外部的 `../../docs-demo` 目录解析并渲染为可交互 Demo；`markdown.config` 中维护 zh-CN / en-US / ja-JP / ko-KR 的语言映射。
- 静态资源统一放在 `public/assets/`，图标与 Logo 通过 `/assets/...` 路径引用。
- 依赖方向：`.vitepress/config.ts` → `./src/config/{zh,en,ja,ko}`（按 config.ts 中的 import 推断存在）→ 各语言 Markdown 页面；Markdown 页面仅依赖外部 `docs-demo` 源码，不反向引用站点配置。
