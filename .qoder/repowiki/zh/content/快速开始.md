## 安装与环境准备

### 包管理器支持
StkTable React 支持所有主流包管理器：npm、yarn、pnpm。

### 安装命令
选择你常用的包管理器执行安装：

```bash
# npm
npm install stk-table-react

# yarn
yarn add stk-table-react

# pnpm
pnpm add stk-table-react
```

### 环境要求
- **Node.js**: 建议使用较新版本（参考 package.json 中的 engines 字段）
- **构建工具**: 推荐使用 Vite，也支持其他现代前端工程化方案
- **TypeScript**: 可选但推荐使用，可获得更好的类型提示

## 基本用法

### 引入组件与样式
在你的 React 组件中导入 StkTable 和样式文件：

```tsx
import { StkTable } from 'stk-table-react';
import 'stk-table-react/lib/style.css';
```

### 定义列配置
创建表格列的配置数组，至少包含标题和数据字段映射：

```tsx
const columns = [
  { title: '姓名', dataIndex: 'name' },
  { title: '年龄', dataIndex: 'age' },
  { title: '城市', dataIndex: 'city' }
];
```

### 提供数据源
准备你的表格数据，每行数据为一个对象：

```tsx
const dataSource = [
  { id: 1, name: '张三', age: 25, city: '北京' },
  { id: 2, name: '李四', age: 30, city: '上海' },
  { id: 3, name: '王五', age: 28, city: '广州' }
];
```

### 渲染表格
将列配置和数据传入 StkTable 组件：

```tsx
function App() {
  return (
    <div style={{ width: '100%', height: '400px' }}>
      <StkTable columns={columns} dataSource={dataSource} />
    </div>
  );
}
```

## 完整示例

### 基础表格示例
以下是一个完整的可运行示例：

```tsx
import React from 'react';
import { StkTable } from 'stk-table-react';
import 'stk-table-react/lib/style.css';

const columns = [
  { title: 'ID', dataIndex: 'id', width: 80 },
  { title: '姓名', dataIndex: 'name', width: 120 },
  { title: '年龄', dataIndex: 'age', width: 80 },
  { title: '城市', dataIndex: 'city', width: 120 }
];

const dataSource = [
  { id: 1, name: '张三', age: 25, city: '北京' },
  { id: 2, name: '李四', age: 30, city: '上海' },
  { id: 3, name: '王五', age: 28, city: '广州' },
  { id: 4, name: '赵六', age: 35, city: '深圳' },
  { id: 5, name: '钱七', age: 27, city: '杭州' }
];

export default function BasicTable() {
  return (
    <div style={{ padding: '20px' }}>
      <h2>基础表格示例</h2>
      <div style={{ width: '100%', height: '400px' }}>
        <StkTable 
          columns={columns} 
          dataSource={dataSource}
          rowKey="id"
        />
      </div>
    </div>
  );
}
```

## 常见问题

### 样式未生效
**问题**: 表格样式显示不正确或默认样式丢失

**解决方案**:
- 确认已正确引入样式文件：`import 'stk-table-react/lib/style.css'`
- 检查构建工具是否能正确解析 CSS 文件
- 确保样式文件的引入顺序在组件之前

### 组件无法渲染或白屏
**问题**: 页面空白或组件不显示

**解决方案**:
- 检查 React 版本是否与包声明兼容
- 查看控制台是否有模块导入错误
- 确认组件容器有明确的宽高设置

### 列不显示或数据为空
**问题**: 表格列名不显示或数据行不渲染

**解决方案**:
- 确认列配置的 `dataIndex` 字段名与数据对象的键一致
- 检查数据是否为空数组或未正确传递
- 验证 `rowKey` 属性是否正确设置

### 本地调试无法启动
**问题**: 开发服务器启动失败

**解决方案**:
- 检查 Node.js 版本是否符合 engines 要求
- 确认依赖是否完整安装
- 检查端口是否被占用
