# 宽高

表格的宽高由根容器的宽高决定。

因此，您可以将表格放置在`flex`容器中，由`flex`管理高度，而不用手动设置宽高。

## 固定宽高

::: tip
通过 `style` 控制表格宽度高度。
:::
